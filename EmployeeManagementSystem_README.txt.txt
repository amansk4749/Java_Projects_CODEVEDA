# Employee Management System in Java

## 📌 Project Overview
This is a simple **Employee Management System (EMS)** built in Java.  
It allows users to **Create, Read, Update, and Delete (CRUD)** employee records using a console-based menu.  

This project was developed as part of my **Codveda Internship (Level 2 - Intermediate Task 1)**.

---

## ⚙️ Features
- **Add Employee**: Add a new employee with ID, name, and salary.  
- **View Employees**: Display all employees in the system.  
- **Update Employee**: Update an employee’s name and salary using their ID.  
- **Delete Employee**: Remove an employee record using their ID.  
- **Exit**: Quit the program safely.

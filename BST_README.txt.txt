# Binary Search Tree (BST) in Java

## 📌 Project Overview
This project is a simple **Binary Search Tree (BST)** implementation in Java.  
It supports the following functionalities:
- Insertion of nodes
- Deletion of nodes
- Traversals (Inorder, Preorder, Postorder)
- Finding the inorder successor when deleting nodes with two children

This project was developed as part of my **Codveda Internship (Level 3 - Advanced Task 3)**.

---

## ⚙️ Features
- **Insert**: Adds a new node in the correct position.  
- **Delete**: Removes a node (handles cases: no child, one child, two children).  
- **Traversals**:
  - Inorder (Left → Root → Right)  
  - Preorder (Root → Left → Right)  
  - Postorder (Left → Right → Root)  
- **Inorder Successor** logic used for deletion when a node has two children.
